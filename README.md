# 🚀 GeoSales AI Dashboard

An AI-powered sales analytics dashboard built using Streamlit and Machine Learning.

---

## 🌐 Live Dashboard
👉 https://geosales-ai.streamlit.app/

---

## 📊 Features

- Interactive sales dashboard
- Region-wise and category-wise filtering
- Monthly sales trend analysis
- Sales forecasting using Linear Regression
- Machine Learning based prediction
- Feature importance visualization

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
